# 🥤 Drink Water Reminder
A simple Python automation tool that sends desktop notifications to help you stay hydrated.

## 🚀 How it Works
The script uses the `plyer` library to trigger system notifications and `time` to manage intervals. It also maintains a `log.txt` file to track your hydration history.

## 🛠️ Setup
1. Clone the repo:
   `git clone https://github.com/your-username/Water-Reminder-Python.git`
2. Install dependencies:
   `pip install plyer`
3. Run the script:
   `python main.py`
